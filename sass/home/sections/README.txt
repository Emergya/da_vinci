Este directorio contiene la maquetacion SOLO de las secciones, no de los contenidos de las mismas.
Si la seccion necesita un padding, o un margen entre contenido y contenido, o posicionar algun contenido, este es el lugar.

Si algun contenido necesita un color de fondo en particular, este NO es el lugar ;)

Por cierto: "seccion" es algo similar a una "region" de drupal.
